#Youtube Bot

Takes User Name , Queary to search, save the logs in the UserName.txt file.
Searches the Query and plays the first video in web browser.

The project is built in 3 modules:
  Logs.py
  User.py
  HTMLScrapping.py
 
Run the bot using User.py.
